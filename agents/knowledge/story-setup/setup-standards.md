# 创作设定协调指南

## 角色层面

- **属性模板**：每角色至少包含 姓名/年龄/身份/外貌/性格/动机/内心冲突/成长弧光
- **关系网**：角色之间至少有"认识"以上的关系（利益/情感/血缘/敌对）
- **知识边界**：每个角色知道什么、不知道什么，必须明确记录

## 剧情层面

- **分卷结构**：每卷有独立的核心冲突和解决
- **伏笔管理**：伏笔埋设时需要登记在 `大纲/伏笔追踪.md`
- **节奏控制**：紧张章节后紧跟放松章节，避免持续高密度冲突

## 世界观层面

- **一致性**：力量体系的规则不能中途改变（除非有合理剧情解释）
- **限制**：角色够强≠可以为所欲为，必须有制约机制
- **成本**：使用力量/魔法/系统必须有代价

## 协调检查清单

- [ ] 角色动机与世界观设定不冲突
- [ ] 剧情发展依赖的世界观规则已在前文铺垫
- [ ] 角色能力增长的节奏与剧情需要匹配
- [ ] 新引入的设定不被后续剧情"遗忘"
- [ ] 不同势力的技术水平符合世界基线
