# 审查官智能体专用知识库

> 本知识库可在每次审查后根据经验优化更新。

## 审查维度权重

| 维度 | 权重 | 说明 |
|:----|:----:|:-----|
| 基础规范 | ★★☆ | 字数/破折号/文件名，基础门槛，不达标直接退回 |
| 叙事结构 | ★★★ | 结构问题会导致读者流失 |
| 角色逻辑 | ★★★★ | 角色OOC是致命伤 |
| 情节质量 | ★★★★ | 是否完成章纲目标、张力是否足够 |
| 语言质量 | ★★★ | AI味、语病影响阅读体验 |
| 标点规范 | ★★☆ | 句号/逗号/破折号的基本功 |
| 设定一致性 | ★★★★★ | 设定矛盾是全书级问题 |
| 系统术语保密 | ★★★★★ | 系统内部概念对外说=世界观崩塌 |

## 常见错误拦截指南

### 必须拦截（高优先级）
| 错误类型 | 示例 | 处理方式 |
|:--------|:----|:---------|
| 设定矛盾 | 角色说了自己不该知道的事 | ❌ 退回，附修改建议 |
| 系统术语对外说 | 陈默对老托马斯说"十二分爬到三十分" | ❌ 退回，改为内心独白 |
| 字数不达标 | <2000或>4000 | ❌ 退回 |
| 破折号超标 | >3处 | ❌ 退回 |
| 角色OOC | 陈默突然变得很勇猛/艾琳话太多 | ❌ 退回 |

### 建议修改（中优先级）
| 错误类型 | 处理方式 |
|:--------|:---------|
| 句号断句生硬 | ⚠️ 建议改为逗号 |
| AI味（过渡词/对称句式） | ⚠️ 建议简化 |
| 动作后接感知用句号 | ⚠️ 建议改为逗号 |
| 心理活动用冒号 | ⚠️ 建议改为逗号 |

### 可改可不改（低优先级）
| 错误类型 | 处理方式 |
|:--------|:---------|
| 措辞可优化 | 💡 可选 |
| 细节增强 | 💡 可选 |

## 拦截优先级速查

遇到问题时按此顺序判断：
1. 这是设定矛盾吗？→ 🔴 必须退
2. 这是系统术语对外说吗？→ 🔴 必须退  
3. 这是字数/破折号问题吗？→ 🔴 必须退
4. 这是角色OOC吗？→ 🔴 必须退
5. 这是结构/情节问题吗？→ 🟡 建议改
6. 这是语言/标点优化吗？→ 🟢 可选
