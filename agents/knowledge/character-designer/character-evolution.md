# 角色设计师智能体专用知识库

> 本知识库可在每次角色更新后根据经验优化更新。

## 角色档案更新规范

### 每次更新必须检查的字段

| 字段 | 是否必须更新 | 说明 |
|:----|:----------:|:-----|
| 当前腐化值 | ✅ 是 | 按推进表逐章更新 |
| 关键事件线 | ✅ 是 | 每条事件一行，含章节号 |
| 心理状态 | ✅ 是 | 因本章事件产生的心理变化 |
| 等级/能力 | ⚠️ 有变化时 | 突破时更新 |
| 关键关系 | ⚠️ 有变化时 | 新建立或破裂的关系 |

## 腐化值推进规律

### 当前小说腐化值节奏
- 第40章末：2%（第一卷末）
- 第70章：12%（左眼紫色圈）
- 第75章：13%（子爵密室共鸣）
- 第81章：15%（第二卷末）
- 每章上涨幅度通常为0~3%

### 腐化值上涨触发器
| 触发条件 | 上涨幅度 |
|:--------|:--------:|
| 接触深渊能量 | 0.5~2% |
| 使用系统能力战斗 | 1~2% |
| 靠近深渊裂隙 | 0.5~1%/次 |
| 击杀深渊生物 | 1~3% |
| 情绪剧烈波动 | 0~1% |
| 长期暴露在深渊环境中 | 0.5%/天 |

## 角色弧光类型

### 常用弧光
| 类型 | 适用角色 | 示例 |
|:----|:--------|:-----|
| 成长弧 | 配角、年轻角色 | 罗姆：怕死少年→战士 |
| 悲剧弧 | 过去已完成 | 维克多子爵：封印者→牺牲 |
| 揭秘弧 | 神秘角色 | 莉莉丝：神秘魔女→前任宿主 |
| 堕落弧 | 反派 | 暂未展开 |

## 文件更新规则

### 每条关键事件格式
```
- **[[第XX章]]**：事件描述（一句话）
```

### 腐化值格式
```
| 当前腐化值 | XX%（第XX章末，状态描述） |
```
