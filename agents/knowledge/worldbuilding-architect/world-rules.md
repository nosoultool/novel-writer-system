# 世界观架构师智能体专用知识库

> 本知识库可在每次设定更新后根据经验优化更新。

## 本世界观核心设定

### 世界格局
- 世界脊梁山脉分东西大陆
- 西大陆：旧帝国封印网络覆盖，深渊被系统性压制
- 东大陆：无封印网络，深渊泄漏常态化

### 力量体系
```
黑铁 → 青铜 → 白银 → 黄金 → 秘银 → 传奇 → 史诗 → 神话
```
- 每阶分初阶/中阶/高阶三档
- 当前主角：青铜初阶

### 深渊能量五条使用途径
| 途径 | 代价 | 使用者 |
|:----|:----|:-------|
| 被动腐化 | 逐步失去自我 | 一切生灵 |
| 系统绑定 | 腐化值上涨 | 陈默、莉莉丝（前任） |
| 圣光转化 | 零表面代价 | 教会神职人员 |
| 封印操控 | 低代价 | 守密者后裔 |
| 意志投降 | 被侵蚀 | 深渊使徒 |

## 设定一致性检查要点

### 技术层面
- 当前时代技术水平：中世纪晚期（铁器+水车+马匹）
- 不存在火器、印刷术、工业机械
- 魔法/深渊能量是已知的超自然力量
- 黯钢是矮人锻造术的特殊产物，稀有

### 社会层面
- 贵族体系：公爵→侯爵→伯爵→子爵→男爵→准男爵→骑士
- 教会势力遍及全大陆，但边境地区影响力薄弱
- 拜伦王国是边境王国，相对落后

### 穿越者知识限制
- 陈默不能在地球知识没过时写出超时代发明
- 他的程序员思维可以优化管理方式，但不能制造科技武器

## 时代背景速查
- 旧帝国覆灭：约5000年前
- 封印网络建成：旧帝国时期
- 圣光教廷创立：约1900年前
- 上一次封印加固：约1900年前
- 月影森林覆灭：约3年前
