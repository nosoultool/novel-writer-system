# 剧情架构师智能体专用知识库

> 本知识库可在每次剧情更新后根据经验优化更新。

## 伏笔管理规范

### 伏笔追踪表格式
```
| 编号 | 伏笔描述 | 埋设章节 | 预期回收 | 优先级 | 状态 |
```

### 优先级分级
| 级别 | 说明 | 处理时限 |
|:----|:-----|:---------|
| ★★★★★ | 全书级核心谜题 | 需要跨卷规划 |
| ★★★★ | 本卷级关键伏笔 | 卷内回收 |
| ★★★ | 角色/情节线伏笔 | 1~2卷内回收 |
| ★★ | 细节/彩蛋级 | 可长期放置 |

### 状态标记
| 状态 | 含义 | 操作 |
|:----|:-----|:-----|
| 待回收 | 已埋设未回收 | 等待合适章节 |
| 进展中 | 已部分展开 | 持续跟踪 |
| 已回收 | 已揭示/解决 | 移入已回收表 |

## 节奏控制

### 张弛节奏（每5章一个周期）
```
紧张章 → 紧张章 → 缓和章 → 爆发章 → 余韵章
示例：
72(紧张) → 73(爆发·罗姆死) → 74(余韵·葬礼) → 75(紧张·子爵) → 76(爆发·子爵终局)
```

### 爽点分布密度
- 本卷（第2卷）主要爽点间隔：每5~8章一个高潮
- 刀片（情感打击）间隔：每8~12章一个
- 暗线揭示间隔：每3~5章一个小揭示

## 大纲一致性检查要点

每次分析时确认：
1. 本章内容与卷纲有无冲突？
2. 本章内容与总纲有无冲突？
3. 角色行为是否符合该阶段的人物弧光？
4. 新埋伏笔是否与已有伏笔表重复？
5. 伏笔回收后是否及时标记？
