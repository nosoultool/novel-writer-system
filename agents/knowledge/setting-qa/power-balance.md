# 力量平衡检查标准

## 核心原则

没有绝对无敌的角色或势力，所有力量都必须有制约。冲突的张力来源于旗鼓相当，或在绝境中寻找胜机。

## 检查清单

### 主角实力定位
- [ ] 主角的成长速度在同类中是否合理？（太快则失去成就感，太慢则失去爽感）
- [ ] 主角的金手指/外挂是否有明确的限制或代价？
- [ ] 主角是否存在"突然变强"的无铺垫提升？
- [ ] 主角的核心优势是否具有不可替代性？

### 反派威胁力
- [ ] 反派是否具备匹配或超越主角的实力/智力/资源？
- [ ] 反派的失败是否不是因为降智，而是合理的弱点被抓住？
- [ ] 是否有多个层次的反派，形成阶梯式威胁？
- [ ] 反派的动机是否足以支撑其行为？

### 配角存在感
- [ ] 配角是否有独立于主角的战斗/决策高光时刻？
- [ ] 配角的能力在团队中是否有不可替代的价值？
- [ ] 配角没有被主角的成长速度完全抛下？

### 势力制衡
- [ ] 主要势力之间的实力对比是否形成了恐怖平衡或动态博弈？
- [ ] 没有哪个势力强到可以碾压所有其他势力（除非刻意设定）
- [ ] 弱势势力是否有独特的生存策略（外交、地理、特殊资源）
