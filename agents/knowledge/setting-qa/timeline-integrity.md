# 时间线完整性检查标准

## 核心原则

时间是最容易被忽略的设定维度，所有事件必须在时间线上找到合理位置。角色状态的变化需要足够的时间支撑。

## 检查清单

### 宏观时间线
- [ ] 世界历史中的重要事件有明确的时间标注（年份/纪元）
- [ ] 关键历史事件之间的间隔合理（文明发展速度符合常识）
- [ ] 不存在"千年不变"的社会制度（除非有设定理由）

### 角色时间线
- [ ] 各角色的年龄在故事开始时是明确的
- [ ] 角色在故事推进过程中的年龄变化被跟踪
- [ ] 角色从出生到现在的经历时间段没有大段空白（或空白有理由）
- [ ] 角色的衰老/寿命设定与种族/修为一致

### 剧情时间线
- [ ] 从故事开始到结束的总时长是明确的
- [ ] 每卷/每章覆盖的时间跨度合理
- [ ] 角色的成长弧光跨越了足够的时间（一夜变强不合理）
- [ ] 事件A和事件B之间的时间间隔支持因果关系的建立
- [ ] 没有时间矛盾——角色同时出现在两个地方

### 季节/天象一致性
- [ ] 关键事件发生的季节/天气与环境描写一致
- [ ] 不同地点的时间流逝同步（除非有时空扭曲设定）
- [ ] 农作物/自然现象的季节性被尊重
