# 逻辑自洽性检查标准

## 核心原则

世界观内部规则必须一致，不能在同一场景下自相矛盾。特殊例外必须有充分的剧情理由。

## 检查清单

### 世界观规则一致性
- [ ] 力量体系/魔法体系的规则在全书范围内保持一致
- [ ] 规则例外情况有明确的前置条件和代价
- [ ] 不同力量体系之间的交互有明确的规则定义
- [ ] 世界的物理/自然规律没有无故改变
- [ ] 特殊能力/血脉/天赋的获得途径和限制清晰

### 势力设定一致性
- [ ] 各势力的目标、立场、行为模式一致
- [ ] 势力之间的关系变化有合理动因
- [ ] 各势力的技术/军事/经济水平与设定匹配
- [ ] 盟约/敌对关系的维系有利益基础

### 设定排他性检查
- [ ] 没有两个设定相互矛盾（如A设定说魔法需要咏唱，B设定说主角瞬发无代价）
- [ ] 没有"一次性"设定（仅在某一章出现后就被遗忘的概念）
- [ ] 没有"万金油"式的解答（一个能力解决所有问题）
