# 因果链检查标准

## 核心原则

每个行为必须有充分的动机支撑，每个后果必须有明确的原因。违反因果关系的巧合必须有前文铺垫。

## 检查清单

### 角色动机检查
- [ ] 每个角色的核心行为都有符合其人设的动机
- [ ] 角色动机的转变有充分的催化事件
- [ ] 没有"为推进剧情而强行降智"的情况
- [ ] 反派做坏事的动机不仅仅是因为坏，而是有利益/信念/经历支撑

### 事件因果链
- [ ] 重大剧情转折都有前文的铺垫（显性或隐性伏笔）
- [ ] 不存在"主角恰好路过"式的机械降神
- [ ] 角色获得关键信息/道具/能力的方式合理
- [ ] 每个冲突的升级有逻辑递进（小冲突→大冲突→决战）
- [ ] 不存在"说了就做到"的跳跃（立下目标后必须有努力过程）

### 设定因果关系
- [ ] 设定A导致设定B，设定B反作用于设定A——形成闭环
- [ ] 设定之间的关系没有断裂（如"主角有检测系统"但"从未用系统检测过关键事件"）
- [ ] 新引入的设定不会否定之前建立的核心矛盾
- [ ] 重大设定的建立和取消都需要代价

### 伏笔因果
- [ ] 伏笔的埋设和回收之间存在合理的因果链条
- [ ] 伏笔回收的时机与剧情节奏匹配
- [ ] 误导性伏笔有合理的"假解释"支撑（不只是为了误导而误导）
