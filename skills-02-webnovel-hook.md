# 🎣 黄金三章/钩子 Skill

## 命令
`/novel:hook`

## 功能
设计网文开篇的黄金三章和高吸引力的钩子。

## 黄金三章原则
- **第1章**：出现危机/矛盾/悬念，让读者想知道"然后呢"
- **第2章**：主角做出选择，展示性格和动机
- **第3章**：小高潮或关键转折，确认追更意愿

## 钩子类型
1. **悬念式**：抛出未解之谜
2. **冲突式**：开篇即展示激烈矛盾
3. **反常识式**：颠覆读者预期
4. **情感式**：引发读者的共情
5. **行动式**：主角立刻采取行动
6. **世界观式**：展示独特有趣的世界

## 平台适配
- **起点**：节奏偏快，金手指早期展现
- **番茄**：节奏极快，每章都有爽点/钩子
- **晋江**：情感铺垫为主，人物先行
