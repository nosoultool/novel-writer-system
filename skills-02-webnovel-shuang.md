# ⚡ 爽点设计 Skill

## 命令
`/novel:shuang`

## 功能
设计网文的爽点（爽感/满足感/期待感）。

## 爽点分类

### 1. 实力碾压
- 扮猪吃老虎
- 越级打脸
- 隐藏实力暴露

### 2. 收获打脸
- 捡漏/奇遇
- 获得稀有资源/能力
- 打脸嘲讽者

### 3. 情感满足
- 忠诚得到回报
- 误会消除
- 付出被认可

### 4. 智斗胜利
- 算计成功
- 布局揭露
- 反转打脸

### 5. 身份揭示
- 隐藏身份曝光
- 真实背景揭露
- 众人震惊

## 爽点分布节奏
- 第1章：小爽（确定基调）
- 每3-5章：中爽（维持期待）
- 每卷：大爽（高潮满足）
- 每隔10-15章：超大爽（重大转折）
