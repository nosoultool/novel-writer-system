# 知识自动分类规则

> 每次产生新的知识沉淀时，按以下规则自动判断归属位置。
> 无需询问，直接放入对应目录。

## 分类判断树

```
新知识产生
  │
  ├─ 是否与具体小说的角色/剧情/设定绑定？
  │   ├─ 是 → 放入当前小说项目
  │   │    ├─ 角色知道/不知道什么 → 小说项目/知识边界/
  │   │    ├─ 角色能力/外貌/关系 → 小说项目/人物/
  │   │    ├─ 剧情走向/伏笔 → 小说项目/大纲/
  │   │    ├─ 世界观规则 → 小说项目/设定集/
  │   │    └─ 已写正文/摘要 → 小说项目/章节/、章节摘要/
  │   │
  │   └─ 否 → 放入灵境系统
  │        ├─ 写作技法/标点/语法 → knowledge/
  │        ├─ 时代背景/历史知识 → .era-knowledge/
  │        ├─ 写作流程/系统机制 → skills/、hooks/
  │        └─ 智能体提示词优化 → agents/
  │
  └─ 不确定时问自己：换一本小说，这条知识还用得上吗？
       ├─ 用得上 → 灵境系统
       └─ 用不上 → 小说项目
```

## 快速判断表

| 知识类型 | 归属 | 示例 |
|:--------|:----|:-----|
| 标点符号用法 | 灵境 `knowledge/` | 句号不能断裂动作与感知内容 |
| 网文写作技巧 | 灵境 `knowledge/` | 爽点设计、钩子类型、节奏控制 |
| 时代背景/历史 | 灵境 `.era-knowledge/` | 中世纪农业技术、封建制度 |
| 写作流程规范 | 灵境 `hooks/`、`skills/` | pre-write流程、字数预算规划 |
| 智能体优化 | 灵境 `agents/` | 写手智能体质量标准更新 |
| **角色知识边界** | **小说项目 `知识边界/`** | 艾琳不知道系统存在 |
| **角色档案** | **小说项目 `人物/`** | 陈默腐化值12% |
| **剧情大纲** | **小说项目 `大纲/`** | 第二卷暗流初涌 |
| **世界观设定** | **小说项目 `设定集/`** | 力量体系、国家关系 |
| **章节正文/摘要** | **小说项目 `章节/`、`章节摘要/`** | 第72章空壳的行动 |
| **审查报告** | **小说项目 `审查报告/`** | chapter_072.review.json |

## 常见误区

| 错误做法 | 正确做法 |
|:--------|:---------|
| 把角色腐化值记在灵境系统 | 记在小说项目 `人物/` 对应角色档案 |
| 把标点规范记在小说项目 | 记在灵境系统 `knowledge/` |
| 把章节正文放在灵境系统 | 放在小说项目 `章节/` |
| 把写作技巧记在小说项目 | 记在灵境系统 `knowledge/` |
