# 描写的经济性原则

## 技法说明
好的描写不在"多"，在"准"。用最少的字数让读者在脑子里生成画面，剩下的交给读者的想象力填充。

## 对比

### ❌ 冗余描写
```
铁锤把罗姆从矿井里抱出来的时候，他的脸上带着悲伤的表情，他的眼睛里含着泪水，他的脚步很沉重，每一步都走得很慢，周围的人看到他这样，也都感到很难过。
```

### ✅ 经济描写
```
铁锤把罗姆从矿井里抱出来的时候，镇上的人已经自己聚集起来了。
```
后者一句写完，不需写铁锤的表情，不需要写人群的反应——"自己聚集起来"五个字已经把该说的都说完了。

## 可复用场景
- 情绪高潮场景：不直接写"他很悲伤"，用周围人的反应或环境暗示
- 战斗场景：不写每一个动作细节，只写最有决定性的一刀
- 对话场景：用动作代替"他说""她说"

## 本项目应用案例
第74章罗姆葬礼的群像写法——不写"悲伤"二字，通过赛娜掩门、柯尔数人、哈克锤落来传递情绪，符合经济性原则。
