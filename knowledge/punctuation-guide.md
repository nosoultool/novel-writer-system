# 网文标点符号用法指南

> 基于国标规范与网文写作实践，经多次写作检验后沉淀的知识。

## 一、核心原则

**标点的作用是让文章意思更明确，而不是机械套规则。** 网文写作中，在清晰传达意思的前提下，优先追求阅读流畅性。

---

## 二、句号使用（最常见的错误）

### 2.1 动作与感知内容的衔接

**规则：** 描述感知动作（看/听/闻/感觉）后紧接感知内容时，用逗号连接，不用句号。

```
❌ 他抬头看去。远处有一个人影。
✅ 他抬头看去，远处有一个人影。

❌ 陈默眯起眼睛才看清。银白色的斗篷。
✅ 陈默眯起眼睛才看清，银白色的斗篷。

❌ 他侧耳听了听。什么声音都没有。
✅ 他侧耳听了听，什么声音都没有。
```

**原理：** 句号会把"看"和"看到的内容"切成两截，读起来像是两个独立事件，而实际上"看清"的行为和"看清的内容"是一体的。

### 2.2 心理活动的衔接

**规则：** 短促的心理活动（计算/回忆/闪过念头）用逗号自然衔接，不用冒号或句号。

```
❌ 他的脑子里在算。三个月。
❌ 他的脑子里在算：三个月。
✅ 他的脑子里在算，三个月。

❌ 他在心里默数。五秒。四秒。三秒。
✅ 他在心里默数，五秒。四秒。三秒。
```

**原理：** 逗号让心理活动融入叙述流，读起来像念头自然滑过。冒号有一种"正式引出"的仪式感，适合关键内心戏，但不适合一闪而过的日常心理活动。

### 2.3 同一对象的补充说明

**规则：** 对同一人物/事物的补充说明用逗号连接，不另起一句。

```
❌ 他脑子里闪过那个白色身影。那也是一个穿银白色斗篷的人。同一个。
✅ 他脑子里闪过那个白色身影，那也是一个穿银白色斗篷的人，同一个。
```

## 三、破折号使用

### 3.1 核心规范

**规则：** 正文中尽量减少破折号（——）。能用逗号、句号、冒号替代的优先替代。

```
❌ 他想到了一个办法——从侧面绕过去。
✅ 他想到了一个办法，从侧面绕过去。

❌ 她看了一眼天空——要下雨了。
✅ 她看了一眼天空，要下雨了。
```

### 3.2 可以保留的场景

| 场景 | 示例 | 说明 |
|------|------|------|
| 声音延长 | "救——命——啊——" | 无可替代 |
| 突然中断 | "你——"他话没说完就倒下了 | 表示话语被打断 |
| 对话语气停顿 | "我——我不是那个意思" | 角色的犹豫/结巴 |

### 3.3 替代方案

| 破折号用法 | 替代方案 |
|-----------|---------|
| 补充说明 | 逗号拆分 |
| 递进 | 用"甚至""更"等词 |
| 转折 | 直接用"但""不过" |

## 四、冒号使用

### 4.1 什么时候用冒号

| 场景 | 示例 | 说明 |
|------|------|------|
| 引出直接引语 | 他问："你叫什么名字？" | 国标规范用法 |
| 提示性话语后 | 他宣布了一个消息：明天放假。 | 正式引出内容 |
| 关键内心戏 | 他在心里呐喊：绝不能放弃！ | 情感爆发点，让人物"站出来说话" |

### 4.2 什么时候不用冒号

| 场景 | 正确写法 | 说明 |
|------|---------|------|
| 日常心理活动 | 他想，今天天气不错。 | 逗号，自然融入叙述 |
| 短促念头 | 他一算，三个月。 | 逗号，流畅衔接 |
| 动作顺承 | 他想了想，好吧。 | 逗号，不是正式提示 |

### 4.3 冒号 vs 逗号对比

| 对比项 | 冒号 | 逗号 |
|--------|------|------|
| 心理距离 | 远，心理活动独立呈现 | 近，心理融入叙述流 |
| 强调程度 | 强，引起读者注意 | 弱，自然带过 |
| 阅读节奏 | 有停顿感、仪式感 | 更流畅、丝滑 |
| 网文适用性 | 关键内心戏、情感爆发点 | 日常心理描写、短反应 |

**一句话：** 冒号让人物"站出来说话"，逗号让人物"悄悄在心里想"。

## 五、引号使用

### 5.1 对话标点

**规则：** 提示语在对话前用冒号，在对话中间/后用逗号。

```
他问："你叫什么名字？"       # 提示语在前 → 冒号
"你叫什么名字？"他问。         # 提示语在后 → 句号
"我叫陈默，"他说，"领主。"    # 提示语在中间 → 逗号
```

### 5.2 内心独白的引号

| 写法 | 示例 | 效果 |
|------|------|------|
| 无引号+逗号 | 他想，也许还来得及。 | 自然融入，适合日常念头 |
| 无引号+冒号 | 他想：也许还来得及。 | 略有仪式感，适合中等长度 |
| 引号+冒号 | 他想："也许还来得及。" | 强调，适合关键内心戏 |

## 六、省略号使用

### 6.1 规范

- 中文省略号用六个点（……），不是三个点（...）
- 省略号后不再加句号、逗号等标点

### 6.2 常见错误

```
❌ "我不知道..."          # 用了三个点
❌ "我不知道。。。。。"   # 用了句号代替
✅ "我不知道……"         # 正确
```

### 6.3 使用场景

| 场景 | 示例 | 说明 |
|------|------|------|
| 话未说完 | "可是……"他欲言又止。 | 犹豫/未完 |
| 沉默 | "……"他什么都没说。 | 无言的回应 |
| 声音逐渐消失 | 那个声音越来越远…… | 渐弱 |
| 意味深长 | 也许，这就是命运吧…… | 留白 |

## 七、常见错误速查

| 错误写法 | 正确写法 | 错误类型 |
|---------|---------|---------|
| 他抬头看去。远处有人。 | 他抬头看去，远处有人。 | 动作与感知断裂 |
| 他在计算：十天。 | 他在计算，十天。 | 冒号代替逗号 |
| 他想到一个主意——从侧面走。 | 他想到一个主意，从侧面走。 | 多余破折号 |
| "我不知道。。。。" | "我不知道……" | 省略号用错 |
| 他心想："是的。"(日常短念头) | 他心想，是的。 | 引号过度使用 |
