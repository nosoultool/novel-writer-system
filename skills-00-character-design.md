# 👤 角色设计 Skill

## 命令
`/novel:characters`

## 功能
创建和管理小说角色。

## 角色设计流程

### 1. 核心角色
- 主角（性格/动机/缺陷/成长空间）
- 主要配角（盟友/对手/恋人/导师）
- 反派（动机/魅力/威胁等级）

### 2. 角色属性
```
名称：
年龄：
身份/职业：
外貌特征：
性格（MBTI/特质）：
核心动机：
内心冲突：
成长弧光：
关键关系：
台词风格：
```

### 3. 关系网络
- 角色关系图（情感/利益/血缘）
- 关系变化轨迹

### 4. 角色管理
- 当前状态追踪
- 实力/能力变化
- 心理状态记录
