# 🚧 质量门禁 Skill

## 命令
`/novel:quality`

## 功能
6 道质量门禁检查，确保每章达标。

## 6 道门禁

### 门禁 1：基础规范
- [ ] 错别字/语病
- [ ] 标点符号使用正确
- [ ] 段落长度适中
- [ ] 章节字数在目标范围内

### 门禁 2：叙事结构
- [ ] 有明确的开端/发展/结尾
- [ ] 视角清晰一致
- [ ] 时间顺序明确

### 门禁 3：角色逻辑
- [ ] 角色行为符合人设
- [ ] 角色对话有区分度
- [ ] 角色情感变化有铺垫

### 门禁 4：情节质量
- [ ] 本章有推进作用
- [ ] 有足够的冲突或张力
- [ ] 结尾有继续阅读的动力

### 门禁 5：语言质感
- [ ] 无 AI 味（参考 anti-ai 清单）
- [ ] 描写具体生动
- [ ] 对话自然

### 门禁 6：一致性
- [ ] 与之前设定无矛盾
- [ ] 与前文衔接自然
