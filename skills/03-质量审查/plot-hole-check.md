# 🕳️ 漏洞检测 Skill

## 命令
`/novel:plot-hole`

## 功能
检测故事中的逻辑漏洞和剧情硬伤。

## 漏洞类型

### 1. 逻辑漏洞
- 因果关系断裂
- 角色行为与动机不符
- 时间线矛盾

### 2. 设定漏洞
- 力量体系失衡
- 地理/时间描述矛盾
- 科技/魔法水平不一致

### 3. 叙事漏洞
- 视角混乱
- 信息前后矛盾
- 重要情节缺乏铺垫

### 4. 角色漏洞
- 性格突变无原因
- 能力忽高忽低
- 关系变化无过渡

## 检测流程
1. 扫描全本或指定范围
2. 标记可疑段落
3. 分析问题类型
4. 提供修复方案
