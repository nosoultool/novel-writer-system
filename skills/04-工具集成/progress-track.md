# 📈 进度追踪 Skill

## 命令
`/novel:progress`

## 功能
查看和管理写作进度。

## 显示内容
```
📊 [项目名称] 写作进度

目标：XXXX 字
已写：XXXX 字 (XX%)
章节：X/XX 章

📝 最近进度
第 X 章：XXXX 字 [日期]
第 X 章：XXXX 字 [日期]

⏳ 待办
- [ ] 第 X 章大纲
- [ ] 第 X 章正文
- [ ] 第 X 章审查

📅 今日目标：XXXX 字
```

## 统计维度
- 总字数
- 日均字数
- 章节完成数
- 预计完本时间
