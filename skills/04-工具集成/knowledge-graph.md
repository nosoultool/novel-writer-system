# 🕸️ 知识图谱管理 Skill

## 命令
`/novel:knowledge`

## 功能
管理小说的知识图谱（实体关系网络）。

## 实体类型
- **角色**：主角、配角、反派、NPC
- **地点**：大陆、城市、宗门、秘境
- **势力**：宗门、家族、国家、组织
- **物品**：神器、法宝、丹药、秘籍
- **事件**：历史大事件、角色关键事件

## 管理操作
- 添加新实体
- 编辑实体属性
- 建立/修改关系
- 删除实体
- 查询关系路径

## 存储格式
JSON 格式，每个实体包含 id、类型、属性、关联列表。
