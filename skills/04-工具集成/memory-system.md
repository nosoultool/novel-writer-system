# 🧠 记忆系统 Skill

## 命令
`/novel:memory`

## 功能
管理写作记忆，确保跨会话的连贯性。

## 记忆分层
1. **项目级记忆**：世界观、角色设定、大纲（长期保存）
2. **写作级记忆**：当前进度、近期写作记录（会话间共享）
3. **会话级记忆**：当前对话上下文（临时）

## 记忆内容
- 已写章节的关键信息
- 角色状态变化
- 伏笔状态
- 写作风格偏好
- 待办事项

## 同步机制
每次 `archive` 操作自动更新记忆。
