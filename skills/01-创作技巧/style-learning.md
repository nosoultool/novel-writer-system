# 📚 风格学习 Skill

## 命令
`/novel:style-learn`

## 功能
分析目标风格/作者的特征，学习并应用到创作中。

## 分析维度
1. **词汇偏好**：高频词、形容词使用、动词力度
2. **句式特征**：长句/短句比例、修辞使用频率
3. **节奏特征**：段落长度、对话比例、描写比例
4. **描写风格**：感官侧重（视觉/听觉/触觉）
5. **情感表达**：直白/含蓄、反讽/真诚
6. **叙事视角**：内聚焦/外聚焦、视角切换频率

## 应用
1. 提供参考文本（可为自己的旧作或喜欢的作者）
2. 系统分析 6 个维度的特征
3. 输出风格配置文件
4. 写手 Agent 在写作时加载并遵循
