# 🧩 解耦写作法 Skill

## 命令
`/novel:decoupled`

## 功能
将角色特质与情节要素解耦，组合出多样化的写作方案。

## 核心概念

### 角色解耦
```
角色 = 性格 × 情绪 × 行动 × 对话
- 性格：勇敢/懦弱/幽默/严肃...
- 情绪：愤怒/悲伤/喜悦/恐惧...
- 行动：主动/被动/配合/反抗...
- 对话：直白/隐晦/激烈/沉默...
```

### 情节解耦
```
章节 = 事件类型 × 风格 × 氛围 × 主导特质
- 事件类型：战斗/对话/探索/日常...
- 风格：紧张/温𣊬/幽默/悬疑...
- 氛围：压抑/明亮/诡异/宁静...
- 主导特质：展现成长/揭示信息/加深关系...
```

## 应用
1. 为每章选择角色特质的组合
2. 为每章选择情节要素的组合
3. 确保每章有独特的组合，避免重复

## 学习模式
分析已有作品，提取其角色×情节的组合模式，应用到自己的创作中。
