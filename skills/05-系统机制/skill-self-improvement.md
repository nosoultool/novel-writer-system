# 🔄 技能自改进机制

## 概述
每个 Skill 在使用后自动记录效果反馈，积累优化经验。下次调用时自动应用改进。

## 流程
每次 Skill 被调用后：
1. 记录调用效果（是否达到预期、用户反馈）
2. 分析可优化点（遗漏了什么、哪里可以更好）
3. 更新 SKILL.md 中的 `### 改进记录` 部分
4. 后续调用时参考改进记录调整行为

## 使用方法
在 SKILL.md 末尾添加改进记录：
```
### 改进记录
- [2026-06-22] 增加了角色一致性检查步骤
- [2026-06-20] 优化了开篇钩子检测，增加黄金三章检查项
```

## 注意事项
- 不修改 Skill 的核心功能
- 只记录"使用经验"而非"永久删除"信息
- 改进记录保留最近 10 条
