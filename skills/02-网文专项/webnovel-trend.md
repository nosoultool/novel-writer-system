# 📊 扫榜/趋势分析 Skill

## 命令
`/novel:trend`

## 功能
分析各平台热门趋势，为创作方向提供参考。

## 分析维度
1. **热门题材**：当前什么类型火
2. **开篇模式**：热门书的开篇结构
3. **爽点密度**：多少字一个小高潮
4. **更新节奏**：趋势字数/更新频率
5. **读者画像**：目标读者的偏好

## 使用方法
1. 提供目标平台和榜单
2. 系统分析 3-5 本热门书的共性
3. 输出趋势报告
4. 结合自身优势选择创作方向

## 注意事项
- 跟踪趋势不等于盲目跟风
- 找到趋势 + 个人特色 = 最佳路径
