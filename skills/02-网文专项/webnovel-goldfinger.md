# 💎 金手指设计 Skill

## 命令
`/novel:goldfinger`

## 功能
设计网文主角的金手指（特殊能力/系统/法宝）。

## 金手指类型

### 1. 系统类
- 签到系统、任务系统、商城系统
- 升级系统、合成系统、抽奖系统

### 2. 重生/穿越类
- 重生携带记忆
- 穿越到异界/游戏/小说
- 预知未来

### 3. 传承/血脉类
- 远古血脉觉醒
- 强者传承
- 神器认主

### 4. 能力类
- 特殊天赋（无限学习/复制/模拟）
- 独特视角（看穿弱点/未来视）
- 特殊空间（储物/修炼/种植）

## 设计原则
1. **有代价**：金手指不能白给
2. **有限制**：避免无脑无敌
3. **有成长**：金手指随剧情升级
4. **有冲突**：金手指自身也是矛盾源
