# 💎 金手指设计 Skill

## 命令
`/novel:goldfinger`

## 功能
设计网文主角的金手指（特殊能力/系统/法宝）。

## 金手指类型

### 1. 系统类
- 签到系统、任务系统、商城系统
- 升级系统、合成系统、抽奖系统

### 2. 重生/穿越类
- 重生携带记忆
- 穿越到异界/游戏/小说
- 预知未来

### 3. 传承/血脉类
- 远古血脉觉醒
- 强者传承
- 神器认主

### 4. 能力类
- 特殊天赋（无限学习/复制/模拟）
- 独特视角（看穿弱点/未来视）
- 特殊空间（储物/修炼/种植）

## 设计原则
1. **有代价**：金手指不能白给
2. **有限制**：避免无脑无敌
3. **有成长**：金手指随剧情升级
4. **有冲突**：金手指自身也是矛盾源

## 能力体系设计进阶

### 核心能力拆解
一个好用的能力体系由多个子能力组成，每个子能力有独立规则：

```
┌─────────────┬──────────────┬──────────────┐
│  子能力 A   │   子能力 B   │   子能力 C   │
│  (扫描)     │   (复制)     │   (应用)     │
├─────────────┼──────────────┼──────────────┤
│ - 消耗      │ - 消耗媒体    │ - 单次/多次  │
│ - 需要接触  │ - 占用存储    │ - 有风险     │
│ - 速度限制  │ - 可被清除    │ - 有副作用   │
└─────────────┴──────────────┴──────────────┘
```

### 规则设计清单
为每个能力定义以下维度：

| 维度 | 说明 | 示例 |
|------|------|------|
| **触发条件** | 什么情况下可以使用 | 需要接触目标、需要视觉锁定 |
| **消耗资源** | 使用能力消耗什么 | 体力、精神力、特殊资源 |
| **使用限制** | 不能做什么 | 一天只能用 N 次、不能对 X 使用 |
| **副作用** | 使用后的代价 | 临时虚弱、记忆混乱、被对方感知 |
| **成长路径** | 如何变强 | 熟练度提升、解锁新用法、突破限制 |
| **反制手段** | 如何被克制 | 特定材料可以阻断、特殊体质免疫 |

### 信息类能力特殊规则
涉及"读取/复制/扫描"信息的能力，需额外定义：
- **存储上限**：能存储多少信息
- **覆盖规则**：新信息是否会覆盖旧信息
- **跨载体传递**：信息能否在不同载体间转移
- **精确度**：复制/读取的信息是否完整
- **跨时间有效性**：过去读取的信息在当下的时间线是否仍然有效

### 常见设计陷阱
- ❌ 能力规则太模糊，导致剧情需要时随意解释
- ❌ 能力太强却没有对应的克制手段
- ❌ 主角滥用能力但代价从未真正兑现
- ❌ 能力设定与世界观基调不匹配
