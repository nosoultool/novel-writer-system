# 📝 投稿/平台适配 Skill

## 命令
`/novel:submit`

## 功能
帮助将小说适配到目标平台并准备投稿。

## 平台适配

### 起点中文网
- 首章需有金手指或特殊设定
- 前3章建立主角和核心冲突
- 推荐字数：2000-3000/章
- 签约要求：3万字左右

### 番茄小说
- 节奏极快，每章至少一个爽点/钩子
- 开头就要抓人
- 推荐字数：1500-2500/章
- 推荐算法友好：完读率重要

### 晋江文学城
- 情感线优先
- 人物刻画重于情节密度
- 推荐字数：3000-5000/章
- 标签准确很重要

## 投稿材料
- 作品简介（50-200字）
- 完整前3章
- 大纲（3万字以上的规划）
- 人物介绍
