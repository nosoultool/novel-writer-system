﻿# 📋 大纲规划 Skill

## 命令
`/novel:outline`

## 功能
规划小说整体结构和章节大纲。

## 流程

### 1. 整体结构
- 三幕式 / 英雄之旅 / Save the Cat / 五幕式
- 核心主题与主线
- 分卷规划

### 2. 分卷大纲
- 每卷核心冲突
- 起承转合
- 关键节点

### 3. 章节大纲
- 每章目标
- 主要事件
- 角色出场/变化
- 伏笔设置

### 4. 节奏规划
- 高潮点分布
- 张弛节奏
- 期待感管理

## 支持的结构模板
- 三幕剧（Setup → Confrontation → Resolution）
- 英雄之旅（12 阶段）
- Save the Cat（15 节拍）
- 黄金三章开篇法

## 高级规划技法

### 1. 长系列节奏控制（20+卷规划）

超长篇网文的关键是**分阶段目标**：

```
第一部：建立世界与角色（1-6卷）
  ├── 角色起源（1-2）
  ├── 第一次考验（3-4）
  └── 重大转折（5-6）

第二部：成长与挑战（7-15卷）
  ├── 新舞台/新敌人（7-9）
  ├── 实力飞跃（10-12）
  └── 最大挫折（13-15）

第三部：决战与终局（16卷+）
  ├── 真相揭露（16-18）
  ├── 最终备战（19-22）
  └── 终极决战（23-26）
```

### 2. 多线叙事规划

| 线型 | 说明 | 示例 |
|------|------|------|
| **主线** | 主角的核心目标 | 变强+保护家人 |
| **感情线** | 人际关系发展 | 与希露菲/艾莉丝/洛琪希 |
| **世界线** | 大背景事件 | 人神/转移事件/龙神 |
| **成长线** | 角色内在变化 | 从自卑到自信 |

**穿插原则**：
- 每章至少推进 1-2 条线
- 不要让某条线断超过 5 章
- 高潮时多线交汇

### 3. 闲话/外传插入机制

闲话的作用：
- **填充时间线**：主角修炼时，配角们也在行动
- **切换视角**：展示主角看不到的关键事件
- **铺垫伏笔**：为主线后续做铺垫
- **调节节奏**：紧张主线后插入轻松外传

插入时机：
```
主线高潮 → 闲话缓冲 → 下一轮铺垫 → 主线推进
```

### 4. 情感高潮布局

情感高潮不是只有战斗和爆炸，还包括：

| 类型 | 说明 | 效果 |
|------|------|------|
| **战斗高潮** | 生死对决 | 紧张刺激 |
| **情感高潮** | 告别/重逢/告白 | 感动落泪 |
| **真相高潮** | 惊天秘密揭露 | 震撼冲击 |
| **成长高潮** | 角色突破自我 | 满足欣慰 |

### 爽点分类
| 类型 | 说明 | 适用场景 |
|------|------|----------|
| **实力碾压** | 扮猪吃老虎、越级打脸 | 战斗/竞技 |
| **收获打脸** | 捡漏奇遇、获得稀有资源 | 探索/冒险 |
| **情感满足** | 忠诚回报、付出被认可 | 情感/羁绊 |
| **智斗胜利** | 算计成功、布局揭露 | 谋略/权斗 |
| **身份揭示** | 隐藏身份曝光、众人震惊 | 身份/背景 |

### 爽点分布节奏
- 每章：小爽（钩子/期待）
- 每3-5章：中爽（维持追更动力）
- 每卷：大爽（高潮满足）
- 每10-15章：超大爽（重大转折）

**布局原则**：每 3-5 章一个小高潮，每卷 1-2 个大高潮

### 5. 伏笔管理

```
埋设（卷1）→ 发展（卷2-3）→ 回收（卷4）
         → 误导（False lead）→ 反转（卷5）
```

伏笔类型：
- **显性伏笔**：明显提示，读者能猜到
- **隐性伏笔**：看似无关的细节，后期才揭示
- **误导伏笔**：引导读者往错误方向想

### 高级伏笔技法

#### 记录/文档式信息释放
通过角色发现的记录/日记/书信逐步释放背景信息：
- 每次释放一个"碎片"，不一次性揭露全部
- 每个碎片本身是完整的局部真相
- 多个碎片组合后才能看到全貌
- 读者可以通过碎片拼图推测，增强参与感

#### 前中期伏笔的后期串联
前期看似无关的多个伏笔，在后期一次性串联回收：
- 伏笔 A（卷1）→ 伏笔 B（卷2）→ 伏笔 C（卷3）
- 卷 5 揭示：A + B + C = 同一个真相的不同侧面
- 这种"串联回收"产生极强的震撼力

#### 信息层设计
每个真相至少有三个层次：

| 层次 | 读者知道 | 主角知道 | 反派知道 | 效果 |
|------|----------|----------|----------|------|
| **表层** | ✅ | ✅ | — | 基本设定 |
| **中层** | ? | ? | ✅ | 制造紧张 |
| **底层** | ❌ | ❌ | ✅ | 终极反转 |

### 6. 时间循环/多周目规划

如果需要设计时间循环类情节：

#### 周目递进原则
```
一周目：发现异常 → 仓促结束
二周目：确认规则 → 收集情报
三周目：开始反击 → 取得进展
四周目：揭示真相 → 应对挑战
...
终周目：最终解决 → 代价显现
```

#### 关键设计规则
- **信息增量**：每轮必须有新信息，不能简单重复
- **差异选择**：主角每轮做出不同选择，导致不同的分支
- **代价递增**：使用循环能力需要付出越来越大的代价
