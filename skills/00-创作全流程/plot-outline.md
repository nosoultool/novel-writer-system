# 📋 大纲规划 Skill

## 命令
`/novel:outline`

## 功能
规划小说整体结构和章节大纲。

## 流程

### 1. 整体结构
- 三幕式 / 英雄之旅 / Save the Cat / 五幕式
- 核心主题与主线
- 分卷规划

### 2. 分卷大纲
- 每卷核心冲突
- 起承转合
- 关键节点

### 3. 章节大纲
- 每章目标
- 主要事件
- 角色出场/变化
- 伏笔设置

### 4. 节奏规划
- 高潮点分布
- 张弛节奏
- 期待感管理

## 支持的结构模板
- 三幕剧（Setup → Confrontation → Resolution）
- 英雄之旅（12 阶段）
- Save the Cat（15 节拍）
- 黄金三章开篇法
