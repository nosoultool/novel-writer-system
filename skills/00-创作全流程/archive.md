# 📦 存档与知识更新 Skill

## 命令
`/novel:archive`

## 功能
将已完成的章节信息更新到项目记忆库，保持知识状态最新。

## 更新内容
1. 角色状态变化记录
2. 已回收/埋设的伏笔
3. 重要事件时间线
4. 世界观新揭示的信息
5. 写作备忘（风格偏好、注意事项）

## 存档格式
```
## [日期] 第X章 存档

### 角色状态
- [角色] → [新状态/变化]

### 伏笔
- ✅ 已回收：[伏笔描述]
- ⏳ 进行中：[伏笔描述]
- 🆕 新埋设：[伏笔描述]

### 关键事件
- [事件] → [影响]

### 写作备忘
- [风格/内容方面的注意事项]
```
