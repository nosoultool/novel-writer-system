﻿# ✍️ 章节写作 Skill

## 命令
`/novel:write`

## 功能
根据大纲和设定写作正文章节。

## 写作前准备
1. 加载章纲：`.story-system/chapters/chapter_XXX.md`
2. 加载详细大纲：`大纲/第N卷《卷名》详细大纲.md` 对应章节
3. 加载角色当前状态：`人物/` 对应角色档案
4. **角色知识边界检查**：查阅当前小说项目的 `知识边界/character-knowledge-boundary.md`，确认本章涉及的角色在当前时间点知道什么、不知道什么。避免角色说出超出其知识范围的信息（如教会审判官不该知道"宿主"概念）
5. **字数预算规划**：将本章内容拆解为 3～5 个段落，估算每段字数，确保总和 ≥ 2200。单场景（如"一封信到来看完想完"）通常不够，需补充至少 1～2 个延伸场景（回忆/环境刻画/多人互动/决策前置细节）
6. 加载前一章正文（确保衔接）
7. 确定本章情感基调
8. 确定视角（POV）

## 章节结构
```
开篇（钩子）→ 展开（冲突/推进）→ 转折/发展 → 结尾（悬念/留白）
```

## 字数规范
- **2000～4000 字**（仅计中文字符，不含标点符号、空格、数字、英文）
- 验证方式（Python）：
  ```python
  import re
  cn = re.findall(r'[\u4e00-\u9fff]', open('文件.md','r',encoding='utf-8').read())
  print(len(cn))
  ```

## 写作规范

### 开篇钩子（Hook）设计
开篇必须在前 200 字内抓住读者，常见钩子类型：

| 类型 | 示例 | 效果 |
|------|-----------------|------|
| **身份反差** | 「34岁无业居无定所」 | 制造好奇 |
| **紧迫情境** | 雨中流落街头 | 引发共情 |
| **金句开场** | 哲理式短句点题 | 营造氛围 |
| **倒叙悬念** | 先展示结果再回溯原因 | 推动翻页 |
| **日常破裂** | 平静生活被突发事件打破 | 制造冲突 |

### 破折号规范
- **正文减少破折号使用**：补充说明/转折/解释等用逗号、句号或冒号替代
- **对话可保留**：角色说话时的语气停顿可用破折号
- **替代方案**：补充说明→逗号拆分，递进→用「甚至」「更」，转折→直接用「但」「不过」

### 叙事结构
```
开篇（200-500字钩子）
  ↓
展开（冲突/推进/内省交替） → 用 ★　★　★ 或 ─── 做场景分隔
  ↓
转折/发展（情感起伏点）
  ↓
结尾（留白/悬念/意外反转）
```

### 专业网文技法

#### 1. 深度内面独白（Internal Monologue）
- **比例建议**：叙述 40% / 独白 30% / 对话 20% / 描写 10%
- **独白技巧**：不是直接说「我很伤心」，而是通过回忆、对比、自我嘲讽来展现情绪
- 例：「我到底做了什么？只是跷掉父母的丧礼，拿没打码的萝莉图片打○枪而已啊……」→ 自嘲式独白，既揭示性格又带黑色幽默

#### 2. 幽默与严肃的平衡（Tone Balancing）
- 悲伤情节后用一句自嘲缓解，不让读者沉浸于压抑
- 严肃对话中插入主角的内心吐槽，制造反差笑点
- 切忌：为搞笑而破坏关键情节的氛围

#### 3. 世界观自然融入（Worldbuilding Integration）
- **规则**：通过角色行动展示世界观，而非旁白解释
- 例：保罗展示剑术时顺便说明战斗力体系，而非单独列一段设定
- **禁忌**：大段纯设定描述打断故事节奏

#### 4. 多视角与闲话（POV Switching & Side Stories）
- 主章用主角 POV，保持代入感
- 「闲话」章切换配角 POV，展示主视角外的关键事件
- 切换时机：主线到自然停顿处，或需要从另一角度推进事件时

#### 5. 情感节奏波形（Emotional Rhythm / 爽点布局）
```    高潮
      /\
     /  \      情感释放
    /    \     /
   / 铺垫 \   /  回落
  /        \ /        下次高潮
 /          \         /
/            \       /
              \     /
               \   /
                \ /
```
- 每章至少一个情感起伏（波峰）
- 情绪积累 3-5 章后安排一次大高潮
- 幽默作为情绪回落（波谷）的缓冲

### 黄金三章原则
- **第1章**：出现危机/矛盾/悬念，让读者想知道"然后呢"
- **第2章**：主角做出选择，展示性格和动机
- **第3章**：小高潮或关键转折，确认追更意愿

### 平台节奏适配
- **起点**：节奏偏快，金手指早期展现
- **番茄**：节奏极快，每章都有爽点/钩子
- **晋江**：情感铺垫为主，人物先行

### 混合写作框架
```
[开篇]    钩子（200-500字）
[铺陈]    场景描写+世界观自然展示（500-800字）
[冲突]    对话+行动推进情节（500-1000字）
[内省]    主角内心独白+情感反应（200-500字）
[转折]    新信息/意外事件打破预期（300-600字）
[结尾]    悬念/留白/情感余韵（200-400字）
```

### 网文场景描写补充

在以叙述/独白/对话/描写比例框架下，**描写**部分请参照以下准则：

#### 避免的写法🚫
- 阳光洒在脸上 / 月光如水 / 寒风刺骨
- 空气中弥漫着紧张的气氛
- 他陷入了沉思 / 他感到一阵悲伤
- 夜深了，四周一片寂静

#### 替代写法✅
- 巴尔的右手一直按在剑柄上，指节泛白
- 呼出的气息在空中凝成白雾，他拢了拢领口
- 他坐在门槛上，把那根草茎咬成了三截
- 他盯着那扇门看了一会儿，然后转身走了

#### 场景锚点技巧
每个场景选一个**感官锚点**来制造记忆点：
- 矿井场景：金属味 / 紫色苔藓 / 火把变色
- 铁匠铺场景：锤声 / 炉火的热浪 / 铁屑的气味
- 领主府场景：油灯噼啪 / 碗碟碰撞 / 老托马斯洗碗的水流声

#### 五感分配速查
| 场景类型 | 优先感官 | 说明 |
|:--|:--|:--|
| 室内对话 | 听觉+视觉 | 光线、声音、小动作 |
| 野外探索 | 嗅觉+触感 | 风、温度、气味变化 |
| 战斗紧张 | 触感+听觉 | 心跳、金属碰撞、呼吸急促 |
| 情感高潮 | 视觉+留白 | 用画面替代直白抒情 |
- 例：「你确定？」他握紧剑柄，指节发白。—— 一个动作代替「紧张地说」
- 每个角色有独特的口头禅/语气/用词习惯
- 对话长度随情绪变化：紧张时短促，放松时绵长

## 写作后检查
- [ ] 字数在2000-4000纯中文字符范围内？
- [ ] 本章完成了章纲中指定的章节焦点？
- [ ] 角色行为是否符合人设？
- [ ] 结尾不是总结式收尾（应有留白/悬念/意外）？
- [ ] 是否有明显的逻辑漏洞？
- [ ] 语言是否自然流畅，无明显AI味？
- [ ] **叙事比例是否合理**？（叙述:独白:对话:描写 ≈ 4:3:2:1）
- [ ] **开篇 200 字是否有钩子**？
- [ ] **是否至少有一处情感起伏**？
- [ ] **世界观是否通过行动/对话展示，而非直接说明**？

## 实战优化记录
- 2026-07：第59-61章多次出现字数不足需事后扩展。建议写作前先用章纲要点估算预期字数（每要点约150-300字），确保初稿命中2000底线。如需事后扩展，优先扩展悬念/情感结尾和对话互动。
