# ✍️ 章节写作 Skill

## 命令
`/novel:write`

## 功能
根据大纲和设定写作正文章节。

## 写作前准备
1. 加载该章大纲设定
2. 加载角色当前状态
3. 确定本章情感基调
4. 确定视角（POV）

## 章节结构
```
开篇（钩子）→ 展开（冲突/推进）→ 转折/发展 → 结尾（悬念/钩子）
```

## 写作规范
- 每章 2000-4000 字
- 开篇要有钩子（悬念/冲突/新信息）
- 每章至少一个情感起伏
- 结尾留有期待感
- 对话符合角色性格
- 描写调动五感

## 写作后检查
- 本章完成了什么功能？
- 角色行为是否符合人设？
- 是否有明显的逻辑漏洞？
- 语言是否自然流畅？
