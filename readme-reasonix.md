### 🤖 Reasonix

[Reasonix](https://reasonix.dev) is an AI coding agent environment. This system is installed as a skill.

```bash
# Direct invocation (available after installation)
/novel-writer I want to write a cultivation novel

# Use specific commands
/novel-writer /novel:world  Help me build a fantasy world
/novel-writer /novel:write  Write Chapter 1
/novel-writer /novel:anti-ai  Remove the AI feel from this text
```

> See [REASONIX.md](REASONIX.md) for detailed usage.
