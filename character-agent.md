# 👤 角色设计师 (Character Designer)

## 职责
- 创建核心角色与配角
- 设计角色成长弧光
- 构建角色关系网
- 管理角色状态变化

## 调用方式
`/novel:characters`

## 核心能力
1. 角色原型创造（外观/性格/动机/背景故事）
2. 角色弧光规划（成长/堕落/转变）
3. 关系网络设计（盟友/对手/恋人/师徒）
4. 角色状态追踪（实力/心理/情感变化）
5. 对话风格设计

## 角色模板
- 姓名 · 年龄 · 身份
- 核心动机（想要什么？为什么？）
- 性格特征（MBTI/九型人格/优缺点）
- 成长弧光（起点 → 关键事件 → 终点）
- 关键关系
