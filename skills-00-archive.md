# 📦 存档与知识更新 Skill

## 命令
`/novel:archive`

## 功能
将已完成章节/卷的信息更新到项目存档，保持知识状态最新。

## 更新触发时机
- 每完成一卷 → 写卷级存档
- 每完成10章 → 增量更新
- 每次角色状态有重大变化时 → 即时记录

## 更新内容
1. 角色状态变化
2. 已回收/新埋设的伏笔
3. 重要事件时间线
4. 世界观新揭示的信息
5. 写作备忘（风格偏好、注意事项）

## 存档格式（卷级）

```markdown
---
tags: [存档]
type: archive
generated: [卷名+范围]
---

# 存档：第N卷《卷名》（第X-Y章）

## 关键事件
- ...

## 角色状态
| 角色 | 卷初状态 | 卷末状态 |

## 伏笔状态
| 伏笔 | 状态 | 说明 |
|:--|:--:|:--|
| ⏳ 进行中 | ... |
| 🆕 新埋设 | ... |
| ✅ 已回收 | ... |
| ⚠️ 半揭示 | ... |

## 写作备忘
- [风格/内容方面的注意事项]

## 数据总览
- 已完成章节数
- 总字数（估算）
- 核心暗线状态
```

## 存档存放
`小说项目/.story-system/archive_卷名.md`

## 参考示例
`.story-system/archive_vol1-2.md` — 第一卷+第二卷实际存档
