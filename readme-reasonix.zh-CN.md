### 🤖 Reasonix

[Reasonix](https://reasonix.dev) 是 AI 编程代理环境，本系统已安装为 skill。

```bash
# 直接调用（安装后即可使用）
/novel-writer 我想写一本修仙小说

# 使用特定命令
/novel-writer /novel:world  帮我搭建一个玄幻世界观
/novel-writer /novel:write  写第一章
/novel-writer /novel:anti-ai  去掉这段文字的AI味
```

> 详细用法见 [REASONIX.zh-CN.md](REASONIX.zh-CN.md)
